  <h1><b>RMDLEdit</b></h1>
  <p><i>A GUI model data editor for Apex Legends Season 3 Models (RMDL v54 subversion 10) for <a href="https://discord.gg/wVsudEruAx">R5Valkyrie</a>.</i></p>
</div>

> [!WARNING]
> **Notice:** RMDLEdit is not complete, issues and other bugs may arise. Please open an issue and be descriptive with your issue and how to recreate it

## Features
- **Editing of RMDL Data:** Allows for editing of data within an RMDL file's that would normally take a hex editor
- **Preview of RMDLS:** Allows for previewing RMDL files
